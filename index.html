JavaScript in Action: Make it Dynamic!

See how a few lines of JavaScript can transform a static page into an interactive experience. You'll write code like this and more:

// Get a reference to an HTML element
const myButton = document.getElementById('myClickButton');
const messageDiv = document.getElementById('message');

// Add an event listener to the button
myButton.addEventListener('click', function() {
    // Change the content of the message div
    messageDiv.textContent = 'Hello, JavaScript World! You clicked me!';
    messageDiv.style.color = '#2a9d8f'; // Change text color
    messageDiv.style.fontWeight = 'bold';
});

// Example of a simple function
function greet(name) {
    return `Welcome, ${name}!`;
}

console.log(greet('Learner')); // Outputs to console: "Welcome, Learner!"
